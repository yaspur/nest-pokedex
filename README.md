# Ejecutar en desarrollo

1. clonar el repo
2. Ejecutar el comando:
```
yarn install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```

## STACK USADO

* MONGO DB
* NEST JS
